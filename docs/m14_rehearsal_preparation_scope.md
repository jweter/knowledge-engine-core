# M14 Preparation Scope Boundary

This branch prepares the controlled rehearsal but does not execute it.

Included:

- operator runbook;
- evidence JSON Schema;
- schema-valid stopped-state example;
- sanitized Markdown report template;
- readiness checklist;
- explicit decision rules;
- regression validation of the evidence contract.

Excluded:

- source PDFs;
- local databases;
- generated rehearsal results;
- fabricated 500-row manifests;
- synthetic performance measurements;
- schema migrations;
- telemetry persistence;
- vector, graph, AI, API, web, or distributed-worker features.

The preparation documents may be merged independently only if their exact-head quality suite and artifact-hygiene review pass. Issue #21 must remain open and blocked until the actual 500-paper entry gate can be satisfied and measured.
